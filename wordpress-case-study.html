ClearTides WordPress Case Study

A collaborative WordPress website project for a coastal conservation concept, built using Agile planning, OpenProject coordination, WordPress development, cPanel database hosting, documentation, and responsive design.

My Role

I played a strong role during the WordPress development stage, including site setup, content integration, responsive layout work, testing, documentation, and project support. I also completed key project documents including the website budget plan.

Tools Used

WordPress, Elementor, Kadence, cPanel hosting/database, OpenProject, Yoast SEO, UpdraftPlus, The Events Calendar, accessibility testing tools, and project documentation.

Project Outcome

The team produced a functional WordPress website concept with donation, volunteer, event, content, and maintenance planning features. The live version is currently being migrated away from the original college hosting environment.

Current Status

This project is currently being rebuilt and migrated from the original college server. I am working to preserve the database structure, rebuild the project, and make the live version available again through my own hosting setup.
